# 🌾 KisanX Backend

KisanX Backend is a RESTful API service built using Node.js, Express.js, and MongoDB. It powers the KisanX marketplace by handling authentication, product management, order processing, notifications, and database operations.

## Features

- JWT Authentication
- OTP Verification
- User Management
- Product Management
- Cart APIs
- Order Processing
- Email Notifications
- Push Notifications
- AI-powered Farmer Assistance

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs
- Brevo SMTP

## API Modules

- Authentication
- Products

- Users
- Notifications
